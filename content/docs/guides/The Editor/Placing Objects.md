## Guide info
Tiny: 1-3 minutes

## TLDR - What this guide covers
- Click and drag in the editor to move around.
- Use the Build tab to place objects, the Edit tab to edit objects, and the Delete tab to remove them.

** **

# 1: :ZoomIn: Navigation

Click and drag in the editor space to move around. You may also use the slider at the top to move around really fast.
:ZoomIn: Use the Zoom buttons to zoom in and out.

<div style="width: fit-content; height: fit-content"><iframe src=https://drive.google.com/file/d/1GkQ0BL28FjrfVG96d67p1OQUxayP-kxI/preview?usp=drivesdk></iframe></div>

# 2: :Block: The Build Tab

This is where you can find objects to place. Objects are organized into lots of different tabs, which you’ll learn more about in later guides. To place an object, click on its button in the Build tab, then click anywhere in the editor space.

<div style="width: fit-content; height: fit-content"><iframe src=https://drive.google.com/file/d/1i1JB6ZIuW1j5iHuIZFrnsQqyliDr_mYH/preview?usp=drivesdk></iframe></div>

:Swipe: To place lots of objects at once, click on the Swipe button to the right. You can now click and drag to place lots of objects. Note that this’ll stop you from moving around in the editor.

<div style="width: fit-content; height: fit-content"><iframe src=https://drive.google.com/file/d/1nr3LGZyVa7LFJ1zsLsUKoxTt8QGggHTS/preview?usp=drivesdk></iframe></div>

# 3: :Arrow: The Edit Tab

This is where you can change an object’s location. Click on any object to **select** it. From here, you can use the :Arrow: arrow buttons to change its location, the :Rotate: rotate & :Flip: flip buttons to rotate/flip it, and the :Scale: scale button to change its size.

<div style="width: fit-content; height: fit-content"><iframe src=https://drive.google.com/file/d/11QZvTc0cTBw2TQZnATs4plPxQLY5NH3A/preview?usp=drivesdk></iframe></div>

:Swipe: If you want to select lots of objects at once, click on the Swipe button.
:FreeMove: :Snap: If you want to drag objects into place, use the Free Move button. While this is enabled, you can use the Snap button to align objects with the grid.
:CopyPaste: If you want to copy and paste the objects you have selected, use the Copy and Paste buttons.
:DeSelect: If you want to stop selecting every object, click on the De-Select button.

<div style="width: fit-content; height: fit-content"><iframe src=https://drive.google.com/file/d/1JSZ296942n9yuLqfPke83z2kek2Ji9Lf/preview?usp=drivesdk></iframe></div>

# 4: :Delete: The Delete Tab

This is where you can remove objects you don’t want. Click on any object to delete it instantly.
:Delete: You can use the trash button to delete any objects you have selected.
