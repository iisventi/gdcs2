{{< callout context="note" title="TLDR - What this guide covers" icon="outline/info-circle" >}}

* An algorithm takes an input, processes it through a precise set of step-by-step instructions, and produces an output. Well-designed algorithms are correct and efficient.
* An algorithm's efficiency is measured using asymptotic analysis, focusing on how well the algorithm scales up to larger input sizes.
* We use abstraction, modularity, and encapsulation to help design good algorithms.
* Algorithms are designed using an iterative process of building, testing, and refining.
